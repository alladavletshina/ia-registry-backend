üìñ Quick Start Guide

  1. Test Authentication: Use Auth Service to login
  2. Get Token: Copy JWT token from login response
  3. Authorize: Click "Authorize" in Swagger UI and paste token
  4. Test APIs: Use Asset Service endpoints with the token
# Test Login (via Gateway) curl -X POST http://localhost:8082/api/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"admin","password":"admin123"}' # Test Asset Service (with token) curl -X GET http://localhost:8082/api/assets/ \ -H "Authorization: Bearer YOUR_TOKEN_HERE"
